# CRM in PC

Легка локальна CRM для персонального ПК. Перший реліз реалізований як статичний вебзастосунок без серверної частини: відкрийте `index.html` у браузері або запустіть локальний HTTP-сервер.

## Можливості першого релізу

- створення власних розділів CRM;
- додавання полів у розділ: текст, число, дата, багаторядковий текст, прапорець;
- позначення полів як обовʼязкових;
- створення, редагування та видалення записів у кожному розділі;
- локальне збереження даних у `localStorage` браузера;
- експорт та імпорт бази у JSON-файл для резервного копіювання або перенесення.

## Запуск

Запустіть локальний HTTP-сервер:

```bash
python3 -m http.server 4173
```

Після цього відкрийте [http://localhost:4173](http://localhost:4173).

## Тести

```bash
npm test
```

Тести перевіряють базові операції сховища: створення розділів, полів, записів, оновлення та валідацію обовʼязкових полів.
