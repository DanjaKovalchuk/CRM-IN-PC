# CRM in PC

Легка локальна CRM для персонального ПК. Тепер перший реліз має **настільний застосунок**: його можна запускати як звичайну програму на компʼютері, без відкриття CRM у браузері.

## Можливості першого релізу

- створення власних розділів CRM;
- додавання полів у розділ: текст, число, дата, багаторядковий текст, прапорець;
- позначення полів як обовʼязкових;
- створення, редагування та видалення записів у кожному розділі;
- локальне збереження даних у JSON-файлі на ПК: `~/.crm_in_pc/data.json`;
- експорт та імпорт бази у JSON-файл для резервного копіювання або перенесення.

## Запуск як програма на ПК

### Windows

1. Встановіть Python 3 з [python.org](https://www.python.org/downloads/) і під час встановлення увімкніть опцію **Add Python to PATH**.
2. Завантажте або розпакуйте цей репозиторій на компʼютер.
3. Запустіть файл `run-crm.bat` подвійним кліком.

### macOS / Linux

```bash
./run-crm.sh
```

Або напряму:

```bash
python3 crm_desktop.py
```

## Як зробити один `.exe` файл для Windows

Якщо потрібен файл, який запускається без командного рядка, зберіть застосунок через PyInstaller:

Найпростіше на Windows запустити подвійним кліком файл `build-windows-exe.bat`. Він встановить PyInstaller і збере програму.

Або вручну:

```bash
python -m pip install pyinstaller
python -m PyInstaller --onefile --windowed --name CRM-in-PC crm_desktop.py
```

Готовий файл буде в папці `dist/CRM-in-PC.exe`.

## Вебверсія

Файли `index.html` та `src/app.js` залишені як легка вебверсія, але основний рекомендований сценарій для персонального ПК — запуск `crm_desktop.py` або `run-crm.bat`.

## Тести

```bash
npm test
python3 -m unittest tests/test_crm_core.py
python3 -m py_compile crm_desktop.py src/crm_core.py
```

Тести перевіряють базові операції сховища: створення розділів, полів, записів, оновлення, JSON-збереження та валідацію обовʼязкових полів.
